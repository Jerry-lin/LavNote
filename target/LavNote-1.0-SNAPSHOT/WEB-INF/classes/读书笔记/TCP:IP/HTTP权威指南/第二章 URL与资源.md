# 第二章 URL与资源

## 2.1概述

URL是浏览器寻找信息是所需要的资源位置。

URL分三个部分

1.URL方案，也就是使用的协议

2.服务器位置

3.资源路径

## 2.2URL的语法

URL最重要的3个部分方案、主机、路径

方案：使用什么协议

主机与端口：那个主机和那个端口

用户名和密码：需要用户输入用户名和密码

路径：资源在服务器那个位置

参数：传递的参数

查询字符串：缩小资源范围

片段：使用“#”来划分，表示资源内部的片段，HTTP服务器只处理整个对象
